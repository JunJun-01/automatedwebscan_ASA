Technical details

About

The Automated Web Scanner Application (ASA) allows users to scan websites for vulnerabilities using different scanning methods. Users can choose between a "Medium" scan and a "Deep" scan. The results of the scans are displayed on a web page, and users can download a detailed report of the scan results.

Key Components:

  1. Flask Web Application: The web application framework that handles user interactions and runs the scan scripts.
  2. Scan Scripts: Python scripts (web_scan.py and web_scan2.py) that perform the actual scanning of the websites.
  3. Docker: Used to containerize the application for easy deployment and consistent environment.

The table below shows the differences between the Medium scan and the Deep scan:

Scanner capabilities Medium scan Deep scan
Duration 20 Minute 1 Hour
Scanning Methods 15 Methods 25 Methods

Warning

The XSS scanner generates HTTP requests which can be flagged as attacks on the server-side (although they are harmless). Do not use it if you don't have proper authorization from the target website owner.

Available Tools

Tools Description
Nmap Nmap (Network Mapper) is a network scanning tool used to discover hosts and services on a computer network. It provides a variety of features for probing computer networks, including host discovery, service discovery, OS detection, and more.
Dnsmap Dnsmap is a subdomain brute-forcing tool that helps in enumerating subdomains of a domain. It is useful for discovering additional subdomains that might not be visible through normal DNS lookups.
Davtest Davtest is a tool to test WebDAV (Web Distributed Authoring and Versioning) servers for various vulnerabilities. It can test for file upload capabilities, command execution, and more.
Nikto Nikto is an open-source web server scanner that performs comprehensive tests against web servers for multiple vulnerabilities. It checks for outdated versions of web servers, version-specific problems, server configuration issues, and more.
Wapiti Wapiti is a web application vulnerability scanner that can detect vulnerabilities such as file disclosure, database injection, XSS (Cross-Site Scripting), and more. It performs "black-box" scans (no source code required).
Wafw00f Wafw00f is a tool that identifies and fingerprints Web Application Firewalls (WAFs). It helps in understanding which WAF a target is using, which can be useful for penetration testing and security assessments.
Uniscan Uniscan is a vulnerability scanner for web applications. It can detect vulnerabilities such as SQL injection, local file inclusion, remote file inclusion, and more by scanning the web application for common security issues.
wget wget is a command-line utility for downloading files from the web. It supports HTTP, HTTPS, and FTP protocols and can download files recursively, making it useful for mirroring websites.
WhatWeb WhatWeb is a tool that identifies websites. Its goal is to answer the question, "What is that website?". WhatWeb recognizes web technologies including content management systems (CMS), blogging platforms, statistics/analytics packages, JavaScript libraries, web servers, and embedded devices.
